Kreiraj račun

Popunite polja kako biste kreirali novi račun

Korak 1 - Tip računa

Tip računa je obavezan.
Podvrsta je obavezna.
Valuta je obavezna.
Tip vlasnika je obavezan.

Korak 2 - Vlasnik

Vlasnik je obavezan.
Početno stanje je obavezno i mora biti 0 ili veće.

Podaci o firmi

Naziv firme je obavezan.
Matični broj je obavezan. Matični broj može sadržavati samo cifre. Matični broj mora sadržavati tačno 8 cifara.
PIB je obavezan. PIB može sadržavati samo cifre. PIB mora sadržavati tačno 9 cifara.
Šifra delatnosti je obavezna.
Adresa je obavezna.
